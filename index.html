Cx is/will be...

A type-safe, interpreted, object oriented language intended to be a subset of C/C++ syntax. Cx intended use is aimed towards system administrative tasks, and network programmability.

#include cxctype
#include cxstdio

int main(){
    char *greet = "Hello, World!";

    greet.each(char c){
        puts(toupper(c).to_str);
    }

    return 0;
}

Supported compilers and platforms

Popular compilers which support C++11 are used to build the Cx interpreter.

  • clang++
  • g++4.7 or later
  • VS 2012 express

Cx language features are restricted to the C++11 standard library and will compile on OSX, Linux, and Windows. Any features deemed platform specific will be implemented in dynamic libraries which can be loaded at run-time.
