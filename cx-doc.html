Language Overview

Cx syntax is borrowed heavily from C++, with a little flavoring from Ruby. Unlike Ruby, Cx does not treat everything as an object. However, any instance of a type (basic or user defined) may include standard member functions. This includes function return values.

The basic language mechanics are currently in development and are subject to change.
